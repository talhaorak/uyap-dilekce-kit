Claude Code ile calisirken once `AGENTS.md`, sonra ilgili `docs/` ve `skills/`
dosyalarini oku.

Kisa ozet:

- Bun kullan.
- Dogrudan `master` uzerinde calisma.
- README ve kullanici dokumanlari Turkce yaz.
- Hukuki yorum, UDF format uretimi ve dilekce rendering katmanlarini karistirma.
- Kod degistiriyorsan test yaz ve `bun test` calistir.
